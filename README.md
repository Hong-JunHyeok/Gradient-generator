# Gradient Generator

🎨 손쉽게 그라데이션을 생성해주는 생성기입니다.

## Stack

- 🟡 Only Vanilla JavaScript (Using TypeScript)
- 🟤 Application Build with Parcel.js
- 🔵 Apply styling using tailwind css
- 💚 Rendering templates using Handlebars

## Dependencies

